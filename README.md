## Diff tool is file(strings) diffchecker 

## Prerequisites

- php >=8.1
- composer

## Install

```sh
composer install
mkdir compare && cd compare
touch file1.txt file2.txt
```

## Usage

Put your content for compare in directory compare:
file1 - original file
file2 - touch file
